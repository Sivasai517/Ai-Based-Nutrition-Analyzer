# NutroDet - Meal Nutrition Analyzer

NutroDet is a Streamlit-based web application that analyzes meal descriptions and provides detailed nutritional feedback.

## Features

- Natural language meal input processing
- Detailed nutrition analysis
- Interactive visualization of nutritional content
- Smart recommendations for balanced meals
- Health tips

## Installation

1. Clone this repository
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

1. Run the application:
```bash
streamlit run app.py
```

2. Open your web browser and navigate to the provided local URL (typically http://localhost:8501)

3. Enter your meal description in the text area and click "Analyze Meal"

## Example Input

"I had 2 boiled eggs, 1 cup of dal, and 1 apple"

## Data Sources

The application uses a curated database of common food items with their nutritional values based on standard serving sizes.
